## Building a project

To build a Monterey.js project, do the following:

<Steps>
  <Step title="Step 1">
    Ensure that the project is initialized, initialize with `monterey init`
  </Step>

  <Step title="Step 2">
    Use the build command:

    ```bash
    monterey build
    ```

    The following command will build the project fully into an output directory
  </Step>
</Steps>

## Build command options

<ParamField path="output" type="string" required={false} initialValue="dist">
  Outputs built `.monjson` files to `.js` files in this directory.
</ParamField>

<ParamField path="minify" type="boolean" required={false} initialValue="false">
  Minifies the output so It doesn't take space
</ParamField>

<ParamField path="sourcemap" type="boolean" required={false} initialValue="false">
  Generates sourcemaps, currently this feature isn't working.
</ParamField>

<ParamField path="transpiler" type="default | online" required={false} initialValue="default">
  Changes the transpiler to a specific or custom transpiler.
</ParamField>
